Position Property Cheatsheet

The position CSS property sets how an element is positioned in a document. The top, right, bottom, and left properties determine the final location of positioned elements.

Position Property Relative to Offsets Special
static normal flow none none
relative itself TRBL none
absolute closest positioned ancestor if any, otherwise the initial containing block. TRBL none
fixed initial containing block TRBL stays in one position regardless of scrolling
sticky nearest scrolling ancestor TRBL once the parent container overflows it sticks in one position
